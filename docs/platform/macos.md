# macOS 平台使用文档

> [!NOTE]说明
> macOS 平台通过侧载 iOS 版专用客户端实现汉化

> [!TIP]提示
> 建议使用 **[Impactor](https://impactor.khcrysalis.dev)** 进行侧载，在终端内键入命令以快速安装：
> ```sh
> brew install --cask impactor
> ```

> [!WARNING]注意
> 专用客户端仅支持 **[Apple Silicon (ARM)](https://support.apple.com/116943)** 机型，不支持 **[Intel Chip (x86)](https://support.apple.com/116943)** 机型

> [!IMPORTANT]重要
> 由于 `PlayCover` 会修改包内文件，通过其安装的客户端无法通过反作弊校验，并会导致客户端闪退
