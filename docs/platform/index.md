# 平台教程

请根据您的设备选择对应平台教程。不同平台的安装方式、资源占用和兼容性要求略有差异，建议先阅读对应页面的说明后再下载。

| 平台 | 推荐对象 | 说明 |
| --- | --- | --- |
| [Android](/platform/android) | Android 手机、平板、模拟器 | 提供共存版和独占版客户端。 |
| [iOS](/platform/ios) | iPhone、iPad | 提供自签版和免签版客户端。 |
| [Windows](/platform/windows) | Windows 客户端用户 | 使用专用启动器启用汉化。 |
| [macOS](/platform/macos) | Apple Silicon Mac 用户 | 通过侧载 iOS 专用客户端使用。 |

> [!TIP] 提示
> 如果您只需要文本汉化，也可以查看各平台页面中的 Kuyo 加速器方案说明。
