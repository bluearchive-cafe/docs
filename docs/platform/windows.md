# Windows 平台使用文档

> [!NOTE]说明
> Windows 平台通过使用 Windows 版专用启动器实现汉化

> [!TIP]提示
> 专用启动器可以覆盖于官方启动器使用，或者独立于官方启动器使用，两者可以共用游戏资源文件

> [!WARNING]注意
> 如遇汉化无法生效，请尝试在系统托盘内彻底退出并重启启动器，然后在启动器设置内点击 <kbd>完整修复</kbd>

> [!IMPORTANT]重要
> 专用启动器下载后解压即可使用，建议覆盖官方启动器文件或移动到固定目录并创建快捷方式
