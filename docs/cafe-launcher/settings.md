# 设置

点击标题栏的齿轮图标进入设置页面。设置按类别分为 7 个标签页：

![设置界面](../assets/screenshots/settings.png)

## 通用（General）

| 设置项 | 选项 | 默认值 | 说明 |
|--------|------|--------|------|
| 语言 | 自动 / English / 简体中文 / 日本語 | 自动 | 界面显示语言，"自动"跟随系统 |
| 主题模式 | 跟随系统 / 浅色 / 深色 | 跟随系统 | 启动器配色方案 |
| 关闭行为 | 最小化到托盘 / 直接退出 | 最小化到托盘 | 点击关闭按钮的行为 |

## 游戏（Game）

| 设置项 | 选项 | 默认值 | 说明 |
|--------|------|--------|------|
| 游戏路径 | 自定义目录 | 启动器同目录 | 游戏安装位置，须以 `YostarGames\BlueArchive_JP` 结尾 |
| 启动检查 | 本地清单 / 远程清单 / 不检查 | 本地清单 | 启动前验证游戏文件的方式 |

## 下载与网络（Download & Network）

| 设置项 | 选项 | 默认值 | 说明 |
|--------|------|--------|------|
| 下载速度限制 | 不限速 / 1 MB/s / 5 MB/s / 10 MB/s / 25 MB/s / 50 MB/s | 不限速 | 限制游戏文件下载速度 |
| CDN 线路 | 官方 / Cafe | 官方 | 游戏文件下载线路 |
| 代理设置 | 直连 / 使用系统代理 | 直连 | 网络代理方式 |

## 外观（Appearance）

| 设置项 | 选项 | 默认值 | 说明 |
|--------|------|--------|------|
| 背景来源 | 内置 / 远程 / 自定义 | 内置 | 启动器背景壁纸 |
| 壁纸填充方式 | 填充 / 等比缩放 / 裁剪填充 | 裁剪填充 | 壁纸缩放策略 |
| 壁纸底色 | 颜色选择器 | 黑色 | 壁纸未覆盖区域的填充色 |
| 主题色模式 | 默认 / 跟随系统 / 取自壁纸 / 自定义 | 默认 | 按钮、进度条等高亮颜色 |
| 自定义主题色 | 颜色选择器 | #2E7DF6 | 当主题色模式设为"自定义"时启用 |

## 通知与内容（Notifications & Content）

| 设置项 | 选项 | 默认值 | 说明 |
|--------|------|--------|------|
| Toast 通知 | 开 / 关 | 开 | 操作结果弹窗通知 |
| 远程内容卡片 | 开 / 关 | 开 | 主界面左侧公告区 |
| 更新通道 | 稳定版 / 测试版 | 稳定版 | 启动器自身更新的版本渠道 |

## 高级（Advanced）

| 设置项 | 选项 | 默认值 | 说明 |
|--------|------|--------|------|
| 日志级别 | Verbose / Debug / Information / Warning / Error / Fatal | Information | 日志详细程度，级别越低记录越多 |
| 资源面板 UID | 文本输入 | 无 | 用于关联资源面板的玩家 UID |
