# 游戏操作

## 安装 / 更新

当检测到新版本时，启动器会自动计算需要下载的文件列表（仅下载变更文件），支持：

- **断点续传** — 下载中断后重新打开可继续
- **暂停 / 恢复** — 随时暂停下载，稍后恢复
- **速度限制** — 在设置中调整下载速度
- **CRC64 校验** — 下载完成后自动校验文件完整性
- **CDN 故障切换** — 主线路不可用时自动切换备用线路

主窗口底部操作区固定在远程内容区下方，不会在 1024×640 最小窗口与内容卡片重叠。安装、进度、暂停控制三种面板由同一操作状态切换。

## 修复

如果游戏文件损坏或缺失：

1. 确保游戏处于"可启动"状态
2. 点击**修复**按钮
3. 启动器对比本地文件与远程清单，重新下载不匹配的文件
4. 最多自动重试 3 次

暂停、继续和停止使用同一个活动操作；停止时会取消传输并按操作意图清理断点状态。重新启动后只会恢复版本、清单、路径和下载源均一致的任务。

## 卸载游戏

1. 打开设置 → 游戏
2. 点击**卸载**并在确认对话框中确认
3. 启动器仅删除本地清单中管理的游戏文件与安装状态

启动器会拒绝在游戏进程运行时卸载，并保留未由清单管理的文件。

## 系统托盘

当关闭行为设为"最小化到托盘"时（默认）：

- 点击窗口关闭按钮 → 最小化到系统托盘
- 右键托盘图标 → 打开窗口 / 退出启动器
- 再次双击桌面快捷方式 → 恢复窗口
