# 使用配置

> 专用客户端首次登录默认不会开启任何汉化功能，并会从官方下载所有资源，需要使用控制面板手动启用汉化功能

## 汉化功能说明

> [!NOTE]说明
> 更改汉化设置需要重新下载或新增下载部分游戏资源，并且需要重启游戏以生效

- 游戏文本（200 MB 重新下载）

  此选项控制绝大多数游戏文本语言，如客户端启用汉化后出现异常可尝试暂时关闭文本汉化

- 主线中配（200 MB 重新下载 + 1 GB 新增下载）

  此选项控制主线故事中的配音语言，作用范围仅限主线故事，不包括大厅语音、活动语音、战斗语音等场景

- 图像视频（2 GB 重新下载）

  此选项控制游戏内图像与视频资源，作用范围包括主线故事、活动剧情、战斗贴图等场景

## 公告内控制面板

> [!NOTE]说明
> 公告内控制面板提供了一种快速便捷且一键即可更改汉化设置的方式

> [!TIP]提示
> 在标题界面点击左下角 <kbd>公告</kbd> 按钮，或在游戏大厅点击左上角 <kbd>公告</kbd> 按钮，即可进入公告并在控制面板内更改汉化设置

> [!IMPORTANT]重要
> 若公告内控制面板使用出现异常，可点击公告内的 <kbd>异常通知</kbd> 标签页，左侧会显示当前 UID，可以使用此 UID 访问浏览器控制面板，建议保存 UID 以备用

## 浏览器控制面板

> [!NOTE]说明
> 浏览器控制面板提供了一种无需进入客户端即可更改汉化设置的方式

> [!TIP]提示
> 首次进入游戏时会有弹窗提醒，此时点击<kbd>前往</kbd>即可跳转到对应自己 UID 的浏览器控制面板，建议保存网页链接以备用

> [!IMPORTANT]重要
> 要访问浏览器控制面板，需要先在公告内控制面板内复制链接，或获得自己的 UID，然后将 UID 拼接到以下 URL 之后：
> ```
> https://control.bluearchive.cafe?uid=
> ```
