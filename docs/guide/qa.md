# 常见问题

> 此处汇总了一些常见问题及其解答，如遇问题可在此处查找解决方案

## 使用汉化会导致封号吗

> 目前没有发生过因使用汉化而导致账号封禁的先例，若担心账号被封请老师自行斟酌是否使用汉化，**[本站不对因使用或无法使用本项目所产生的损失承担任何责任](https://github.com/bluearchive-cafe/bluearchive-cafe?tab=AGPL-3.0-1-ov-file "免责声明")**

## 进入场景时卡住 / 闪退 / 游戏贴图异常

> 游戏资源损坏或需要更新，请老师尝试在标题界面左下角点击 <kbd>菜单</kbd>-<kbd>检查游戏资源完整性</kbd>

## 需下载最新版本客户端

> 客户端版本需要更新，请老师在本站重新下载专用客户端并覆盖安装
